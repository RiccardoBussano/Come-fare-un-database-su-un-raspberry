Installare database mysql/mariadb su Raspberry pi

In questo articolo vedremo come scaricare un database MySQL su un Raspberry Pi. Per riprodurre la guida necesssitiamo di un raspberry con sistema operatvo Raspian installato e l'SSH abilitato su nostro raspberry (Vai all'articolo).

Installare phpmyadmin su Raspberry pi

Phpmyadmin è molto utile perchè permette di gestire i database mysql con interfaccia grafica, molto più pratico rispetto a gestirlo con il terminale (Vai all'articolo).

Usare il Database fuori dalla tua rete locale

Usare il data base fuori dalla nostra rete locale può servire per ad esempio per colegarlo ad un host remoto in questo articolo vedremo come, questo puorterà ad una minore sicurezza al data base (Vai all' articolo).

Impostare il DNS dinamico

Creiamo un HostName

Per creare un host name useremo un tool gratis che si chiama No-IP, oltre al host name, creeremo un DNS dinamico per comodità.

Quindi rechiamoci su No-IP, creiamo un account e effeuiamo il login.

Ricordo che nel piano gratis e possibile creare solo fino a 3 host name per account che devono essere aggiornati ogni 30 giorni opure andranno in ibernazione

Una volta effetuato il login troveremo una scheda simile a questa

Nella sezione Hostname scegliamo un host a piacere, invece nella sezione dominio possiamo scegliere uno tra quelli elencati.
Poi clicchiamo su aggiungi HostName.

Installiamo DUC

Aggiunto l'HostName cerchiamo nalla barra laterale DNS Dinamico > Dynamic Update Client, qui troveremo il link per l'installazione di DUC (Dynamic Update Client).
Quindi selezionamo la versione per il nostro sistama operativo e scarichiamo il file.

Apriamo il file di installazione, e installiamo DUC.

Impostiamo il DNS dinamico

Una volta instalato corretamente apriamolo, DUC chiedera di effetuare il login con l'account di No-IP che abbiamo creato in precedenza.
Dopo al login ci chiedera di selezionare gli host name su cui vogliamo che DUC imposti il DNS dinamico.

Ora abbiamo il DNS dinamico impostato sul HostName, ogni 5 minuti DUC controllerà se IP del modem è cambiato è lui lo agiornerà automaticamente. Se ne abbiamo bisogno possiamo anche aggiornarlo autmaticamente ciccando Refresh Now

Bene ora il nostro data base e pronto per essere usato, ora alposto di mettere IP del nostro modem e quindi modificarlo ogni volta alle sue variazioni ci basterà inserire il DNS e non dovremo più modificarlo.
