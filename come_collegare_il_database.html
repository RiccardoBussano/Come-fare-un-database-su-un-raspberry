Installare database mysql/mariadb su Raspberry pi

In questo articolo vedremo come scaricare un database MySQL su un Raspberry Pi. Per riprodurre la guida necesssitiamo di un raspberry con sistema operatvo Raspian installato e l'SSH abilitato su nostro raspberry (Vai all'articolo).

Installare phpmyadmin su Raspberry pi

Phpmyadmin è molto utile perchè permette di gestire i database mysql con interfaccia grafica, molto più pratico rispetto a gestirlo con il terminale (Vai all'articolo).

Usare il Database fuori dalla tua rete locale

Usare il data base fuori dalla nostra rete locale può servire per ad esempio per colegarlo ad un host remoto in questo articolo vedremo come, questo puorterà ad una minore sicurezza al data base (Vai all' articolo).

Come collegare il database

Creiamo un file .js

In questo esempio colegeremo il database ad un file .js (javascript).

Creiamo un file di nome index.js in una cartella qualunque. Apriamo un qualunque terminale e spostiamoci nella cartella appena creata scrivendo:
cd nome_directory
Una volta dentro la cartella scriviamo sempre sul terminale
npm init
Ci chiederà diverse informazioni che verranno scritte in un file .json, premiamo sempre invio (almeno che non si voglia modificare qualcosa) è alla fine quando ci verra chiesto scriviamo
yes

Installazione dipendenza MySQL

Nel terminale sempre nella cartella dove risiede il file index.js scriviamo:
npm install mysql
e attendiamo l'installazione

Scriviamo i dati per la connesione

Nel file index.js dobbiamo importare la dipendeza mysql che abbiamo installato in precedenza e scrivere i dati di connesione al database, come si può vedere in figura:

Quindi creiamo la pool mysql dove dovremmo inserire diversi dati.

  • host: inseriamo il ddns che abbiamo creato in precedenza
  • port: 3306
  • user: In questo campo dobbiamo inserire l'utente con cui dovremo effetuare il login, non possiamo usare l'utente root perché ha un ip di accesso in localhost quindi solo dal raspberry ci potremmo collegare (Create un utente o con Ip della macchina su cui hostate il codice, oppure inserite il carattere % durente la creazione del utente su phpmyadmin, qualunque ip sara accetato)
  • password: inserite la password del utente, (per sicurezza, sopratutto se il codice è open source impostate la password nelle variabili di configurazione del server che farà da host al vostro codice
  • database (non obligatorio): scrivete il nome del database per effetuare una connesione speciale

Ora il vostro database è impostato sul file index.js quindi pronto per raccogliere dati in autonomia
