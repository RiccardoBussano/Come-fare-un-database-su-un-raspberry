Installare database mysql/mariadb su Raspberry pi

In questo articolo vedremo come scaricare un database MySQL su un Raspberry Pi. Per riprodurre la guida necesssitiamo di un raspberry con sistema operatvo Raspian installato e l'SSH abilitato su nostro raspberry (Questo articolo).

Phpmyadmin è molto utile perchè permette di gestire i database mysql con interfaccia grafica, molto più pratico rispetto a gestirlo con il terminale (Vai all'articolo).

Installare database mysql/mariadb su Raspberry pi

Come installare mysql/mariadb su apache e gestire un database.

Come per Apache la procedura da seguire dipende dalla versione di Raspbian installata.
Segui quindi la guida relativa alla versione di Raspbian presente nel tuo Raspberry:

Per verificare la versione correntemente installata il comando è il seguente:
cat /etc/os-release

Raspbian Jessie e versioni precedenti (8-)

Digitiamo da terminale i seguenti comandi:
sudo apt-get update
sudo apt-get install mysql-server mysql-client php5-mysql
Mysql è installato.
Il file di configurazione si trova su: /etc/mysql/my.cnf

Raspbian Stretch e versioni successive (9+)

Sulle versioni successive alla 9 di Raspbian, mysql è stato sostituito da mariadb, vediamo quindi come installarlo.
Digitiamo da terminale i seguenti comandi:
sudo apt-get update
sudo apt install mariadb-server mariadb-client php-mysql
Il database è installato, sarà utilizzabile come fosse mysql.
Il file di configurazione si trova su: /etc/mysql/my.cnf

Creazione utente root mysql

Una volta installato mysql, bisogna creare un utente con cui accedervi.
Digitiamo quindi da terminale:
sudo mysql_secure_installation
e seguiamo la procedura guidata che ci permetterà di impostare una password.

Accesso a mysql da terminale

Per accedere a mysql da terminale basta digitare il seguente comando:
sudo mysql

Passaggi successivi

Utilizzare mysql da terminale non è molto pratico, perciò consiglio di installare phpmyadmin per poterlo gestire graficamente.
